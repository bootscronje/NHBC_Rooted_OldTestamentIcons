# GitHub Pages Setup Instructions

Your flashcard viewer is ready to be shared! Follow these steps:

## Quick Setup (2 minutes)

1. **Push your changes to GitHub**
   ```bash
   git add -A
   git commit -m "Add flashcard HTML viewer"
   git push origin main
   ```

2. **Enable GitHub Pages**
   - Go to: https://github.com/bootscronje/NHBC_Rooted_OldTestamentIcons/settings/pages
   - Under "Source", select branch: **main**
   - Under "Branch", select folder: **/ (root)**
   - Click **Save**

3. **Wait 1-2 minutes** for GitHub to deploy

4. **Your shareable link will be:**
   ```
   https://bootscronje.github.io/NHBC_Rooted_OldTestamentIcons/
   ```

5. **Share this link** with anyone - no technical knowledge needed! They just click and use the flashcards.

## What They'll See

- Beautiful interactive flashcards
- Click to flip cards
- Navigation buttons
- Shuffle feature
- Progress tracking
- Works on any device (phone, tablet, computer)
